# AI Customer Support Website

An open-source AI-powered customer support website for businesses.

## Project Goal

This project is designed to help businesses manage customer enquiries using an AI chatbot with live human takeover.

## Features

- Responsive Landing Page
- AI Customer Support
- FAQ Automation
- Human Live Chat Handoff
- WhatsApp Integration
- Contact Form
- Admin Dashboard (Coming Soon)

## Services

- Foreign TikTok Account Creation
- TikTok Ads
- CAC Registration
- Business Consultation

## Tech Stack

- JavaScript
